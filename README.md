**Connectionist Neurons and Neural Network Gradient Derivations**

This repository focuses on the fundamental principles of artificial neural networks, exploring binary classification tasks using single connectionist neurons and the mathematical foundations of error backpropagation in multi-layer perceptrons (MLPs).

**Key Features & Explorations**

Single Neuron Binary Classification: Implemented a connectionist neuron with a binary transfer function to classify non-linearly separable data (Apples vs. Oranges dataset). Analyzed decision boundaries, calculated classification error rates, and evaluated the impact of weight and threshold variations.

Multi-Layer Perceptron (MLP) Error Analysis: Investigated a network architecture with two input units, a hidden layer with three logistic neurons, and a single logistic output neuron to fit continuous target functions.

Mathematical Derivative Proofs: Derived full analytical proofs for the derivatives of the standard logistic sigmoid activation function, showing its elegant relationship to its own output.

Gradient Calculation and Backpropagation: Calculated explicit gradient formulas for the network's error function with respect to both output-layer and hidden-layer weights, demonstrating how error signals propagate backward through the network.
